# Random Password Generator

A simple browser-based tool that generates random passwords. Built with vanilla HTML, CSS, and JavaScript.

## Features

- Generates two password options at once
- Choose your password length
- Optionally include numbers and/or symbols (letters are always included)
- Click a generated password to copy it to your clipboard, with a "Copied!" confirmation

## Demo

Open `index.html` in your browser — no build step or dependencies required.

## Usage

1. Enter a desired password length (minimum 8)
2. Check "Include Numbers" and/or "Include Symbols" if desired
3. Click "Generate Passwords"
4. Click either generated password to copy it

## Tech Stack

- HTML
- CSS (Google Fonts: Karla)
- JavaScript (no frameworks or libraries)
